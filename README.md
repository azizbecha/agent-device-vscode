# Agent Device for VS Code

Author, run, and inspect [agent-device](https://incubator.callstack.com/agent-device) `.ad` scripts inside VS Code.

> Status: scaffold only. POC roadmap drives the build-out — Phases 0–5.

## Development

```bash
bun install
bun run watch
```

Then press `F5` in VS Code to launch the Extension Development Host.

## Package

```bash
bun run package    # produces a .vsix
```
